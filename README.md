# AdminMPA 

## 🛠 개발환경
- Java 11
- SpringBoot 2.7.10
- MariaDB
- MyBatis
- IntelliJ IDEA Ultimate Edition

## 📓 프로젝트 설명
- 클린코드를 지향한 코드작성
- RESTful API 규약에 맞춘 게시판 CRUD 구현
- 코드의 진행사항과 유지보수를 위한 Javadoc 주석 첨부
- 사용자 댓글 구현
- 에러핸들링 및 예외처리
- 일관된 응답 형식에 따른 API 응답 구현(ApiResponseDTO)
- 다중 파일 업로드, 파일 다운로드 구현
